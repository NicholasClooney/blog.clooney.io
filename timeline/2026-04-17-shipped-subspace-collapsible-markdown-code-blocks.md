Shipped GitHub-style collapse and expand controls for long Markdown code blocks in [v1.27.0](https://github.com/TheClooneyCollection/11ty-subspace-builder/releases/tag/v1.27.0) of 11ty-subspace-builder. The release also reuses that shared copy-and-collapse behavior for regular fenced code blocks, which keeps the interaction consistent instead of treating Markdown blocks as a separate case.
